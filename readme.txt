Browser: Chrome
Libraries: Jquery, Handlebars, Query-String, Bootstrap

Files:  js folder contains following files:

config: Contains application level config & api url for making is more configurable
handlers: Contains UI handlers which connects UI to Services
services: Contains BlogService which handles all the api calls and CacheService maintains the app level cache.
utils: Contains Api Utils which uses fetch for making api calls and some global utils
views: Contains all views required in the app
main.js: Entrypoint


Instructions to run:
1. npm install
2. npm start
